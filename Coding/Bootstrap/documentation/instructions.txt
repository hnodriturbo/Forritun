Running documentation locally

1.  Run npm install to install the Node.js dependencies, 
    including Hugo (the site builder).

2.  Run npm run test (or a specific npm script) to rebuild 
    distributed CSS and JavaScript files, as well as our docs 
    assets.

3.  From the root /bootstrap directory, run npm run docs-serve 
    in the command line.
